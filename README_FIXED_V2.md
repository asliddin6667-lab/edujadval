# Edujadval v2.1 — To'liq tekshirilgan paket

## Ishga tushirish

```cmd
npm.cmd install
npm.cmd run dev
```

Brauzerda terminalda chiqqan manzilni oching: `http://localhost:5173/` yoki `http://localhost:5174/`.

## Tuzatilganlar

- Sidebar logotip `public/logo.png` orqali chiqadi.
- Menu yozuvlari active holatda ham ko'rinadi.
